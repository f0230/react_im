# Handbook de Responsabilidades y Precios por Tipo de Tarea

> Documento de referencia para equipo y ventas. Define qué incluye cada tarea, sus entregables, responsabilidades y **rangos de precios para cotización**.

## 💰 Notas sobre Precios

- **Rangos en USD** orientados al mercado uruguayo/LATAM
- **Precios son referenciales** para cotización inicial
- **Factores que afectan el precio final:**
  - Complejidad específica del proyecto
  - Volumen de trabajo (descuentos por cantidad)
  - Tipo de cliente (PYME vs Enterprise)
  - Urgencia/tiempos de entrega
  - Relación con cliente (nuevo vs recurrente)
- **No incluye:** Inversión en pauta publicitaria (media spend)

---

## 📋 Cómo usar este documento

| Rol | Uso |
|-----|-----|
| **Ventas** | Para armar cotizaciones rápidas sumando tareas |
| **Workers** | Para entender el valor comercial de su trabajo |
| **Project Manager** | Para justificar precios al cliente |
| **Approver** | Para validar que el trabajo corresponde al tipo de tarea |

---

---

## 📋 Cómo usar este documento

- **Worker**: Antes de loguear una tarea, revisá si las responsabilidades coinciden con lo que hiciste.
- **Approver**: Usá esto para validar que el trabajo logueado corresponde al tipo de tarea seleccionado.
- **Scope**: Si una tarea incluye varias de estas responsabilidades, podés loguear múltiples tipos de tarea o usar "Trabajo especializado custom" con descripción detallada.

## 🎯 Paid Media

### Meta Ads

#### `meta_business_manager_setup` - Setup Meta Business / activos (5 pts)
**💰 Precio al cliente:** $150 - $300 USD  
**Responsabilidades:**
- Creación/configuración de Business Manager
- Vinculación de páginas de Facebook e Instagram
- Configuración de cuentas publicitarias (límites de gasto, métodos de pago)
- Asignación de permisos y roles a usuarios
- Configuración de dominios verificados
- Vinculación de Instagram Shopping (si aplica)

**Entregables:**
- Business Manager funcional y accesible
- Usuarios con permisos correctos asignados
- Página de pagos configurada

**No incluye:**
- Instalación de pixel (es otra tarea)
- Creación de campañas publicitarias
- Contenido orgánico para las páginas

## 📌 Cómo loguear tareas correctamente

### Si tu trabajo cubre múltiples tareas

**Opción A - Loguear múltiples tareas:**
```
Work log 1: meta_ads_campaign_setup (Configuración campaña Meta Ads)
Work log 2: meta_ads_creative_setup (Setup creatividades Meta Ads)
Work log 3: meta_ads_tracking_setup (Setup pixel / eventos / CAPI)
```

**Opción B - Usar "Trabajo especializado custom" con descripción detallada:**
```
Tipo: custom_specialized_delivery
Descripción: "Setup completo de campaña Meta Ads incluyendo estructura, 
creatividades y tracking. Aprox 8 horas de trabajo total."
Puntos override: 12 (suma de setup + creatividades + tracking)
```

### Criterios para elegir

- **Opción A** es preferible cuando:
  - Las tareas se pueden separar claramente
  - Querés trackear cuánto tiempo lleva cada tipo de trabajo
  - Algunas partes podrían hacerse por diferentes personas

- **Opción B** es preferible cuando:
  - El trabajo es indivisible
  - Es más rápido loguear una sola vez
  - No necesitás granularidad para análisis

### Ejemplos de combinaciones comunes

| Situación | Tareas sugeridas |
|-----------|-----------------|
| Lanzar campaña Meta Ads completa | meta_ads_campaign_setup + meta_ads_creative_setup + meta_ads_tracking_setup |
| Setup inicial de email marketing | email_platform_setup + email_template_design + email_automation_flow |
| Rebranding completo | strategy_rebranding_strategy + design_brand_identity + design_brand_guidelines |
| Landing page end-to-end | web_landing_page_design + copy_website_landing + web_custom_frontend |
| Video comercial completo | video_production_coordination + copy_script_video + video_ad_commercial_edit |

## ⚠️ Reglas de oro

1. **Sé específico**: Si hiciste "optimización", especificá qué se optimizó
2. **No dupliques**: Si ya logueaste la tarea A, no la incluyas en la descripción de la tarea B
3. **Documenta**: Si usás "custom", la descripción es obligatoria y debe ser clara
4. **Preguntá**: Si no estás seguro qué tarea corresponde, consultá con el approver antes de loguear
5. **Sé honesto**: No loguees más puntos de los que corresponde por el trabajo real
