# Handbook de Tareas Simplificado DTE

> Catálogo simplificado por roles y responsabilidades para compensación justa.

---

## 🎯 Estructura General

Cada tarea está organizada por:
- **Rol** (quién lo hace)
- **Nivel** (responsabilidad y expertise requerido)
- **Puntos** (valor para compensación)

---

## 💰 Paid Media / Performance

### Setup Inicial (trabajo puntual, alto valor técnico)

#### `perf_meta_setup` - Setup Meta Ads (6 pts)
**Rol:** Performance Specialist
**Responsabilidades:**
- Crear/configurar Business Manager
- Instalar y configurar Pixel + CAPI básico
- Estructura de campañas, conjuntos, anuncios
- Configuración de audiencias iniciales
- Primeros anuncios publicados

**No incluye:**
- Diseño de creatividades (eso es Design)
- Redacción de copy extenso (eso es Copywriting)
- Optimización mensual (eso es Gestión)

**Entregable:** Cuenta de Meta Ads lista para lanzar campañas.

---

#### `perf_google_setup` - Setup Google Ads (6 pts)
**Rol:** Performance Specialist  
**Responsabilidades:**
- Configuración de cuenta y facturación
- Campaña Search con estructura de grupos de anuncios
- Keywords, match types, negative keywords
- Configuración de conversiones básicas
- Extensiones de anuncios

**No incluye:**
- SEO (eso es Web)
- Landing pages (eso es Web)
- Gestión mensual

**Entregable:** Campaña de Google Ads activa.

---

#### `perf_email_setup` - Setup Email Marketing (5 pts)
**Rol:** Performance Specialist / CRM Specialist
**Responsabilidades:**
- Configuración de plataforma (Mailchimp, Klaviyo, etc.)
- Autenticación de dominio (SPF, DKIM)
- Diseño de templates base
- Primeros flujos automatizados (welcome, etc.)
- Importación y segmentación inicial de lista

**No incluye:**
- Copy de emails (eso es Copywriting)
- Diseño gráfico complejo (eso es Design)
- Gestión mensual de envíos

**Entregable:** Plataforma de email lista para usar.

---

#### `perf_meta_management` - Gestión Meta Ads (5 pts)

- Análisis de métricas y reporting

**Requiere:** Campañas previamente configuradas.

**Entregable:** Campañas optimizadas + reporte mensual.

---

#### `perf_google_management` - Gestión Google Ads (5 pts)

**Entregable:** Campañas optimizadas + reporte.

---

#### `perf_email_management` - Gestión Email Marketing (4 pts)

**Responsabilidades:**
- Creación y envío de newsletters

- Segmentación de listas

**Entregable:** Emails enviados + reporte mensual.

## 📌 Reglas de Oro para Loguear

1. **Elegir la tarea más cercana** - Si no hay match exacto, usar Custom.
2. **No duplicar** - Una tarea logueada no se repite en otra descripción.
3. **Ser específico en la descripción** - "Setup Meta para cliente X" no "Trabajo en Meta".
4. **Custom con criterio** - Solo si realmente no aplica ninguna otra.
5. **Cuando dudes, preguntá** - Mejor aclarar antes que loguear mal.
