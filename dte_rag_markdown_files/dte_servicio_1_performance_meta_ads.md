# Servicio 1 – Campañas de performance (Meta Ads) y optimización

El servicio de Campañas de Performance en DTE está diseñado para generar demanda real: leads, consultas calificadas, reservas o ventas, según el modelo de negocio. No se trata solo de “poner anuncios”, sino de construir un sistema de adquisición con estructura de test, aprendizaje y optimización continua. Trabajamos especialmente fuerte en Meta (Facebook/Instagram) porque, bien ejecutado, sigue siendo uno de los canales más eficientes para escalar en Uruguay y LATAM.

El servicio comienza con un diagnóstico: objetivo del cliente, oferta, públicos, competencia, creatividades disponibles, histórico de campañas (si existe) y capacidad operativa para atender leads. A partir de eso se define una estrategia: tipo de campaña (leads, mensajes, conversiones), eventos a medir, estructura (ABO/CBO según caso), y plan creativo. En DTE cuidamos que las campañas no dependan de una sola idea: se preparan múltiples ángulos de comunicación para probar (dolor, deseo, prueba social, autoridad, oportunidad, urgencia), y se iteran con base en datos.

Un componente clave es el tracking: se configura píxel, eventos, UTMs, y medición de resultados. Si el negocio lo requiere, se arma un circuito de atribución simple pero útil: qué anuncio trajo el lead, qué porcentaje responde, cuántos agendan, cuántos compran. Esto es vital para no optimizar “a ciegas”.

La gestión incluye: creación y carga de campañas, monitoreo diario o frecuente, optimización por rendimiento (costos, CTR, CPA, calidad de lead), control de presupuesto, y reporte periódico con decisiones claras. Cuando el cliente no tiene equipo creativo, DTE puede producir las piezas o guiar la producción (briefs, guiones, edición, UGC).

Entregables típicos: estructura de campañas activa, set de anuncios en test, mejoras iterativas semanales, reportes con insights y próximos pasos. Este servicio funciona mejor cuando se complementa con landing y automatización de atención, pero también puede ejecutarse con flujos existentes del cliente.
