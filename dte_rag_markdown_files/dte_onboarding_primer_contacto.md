# Proceso de onboarding – alta de usuario y primer proyecto en DTE Platform

El onboarding de DTE combina diagnóstico comercial con activación operativa en DTE Platform. El objetivo es que el cliente no quede “a la espera”, sino con cuenta activa, proyecto creado y próximos pasos claros.

Flujo recomendado de onboarding:
1. Primer contacto y diagnóstico corto (objetivo, urgencia, alcance).
2. Alta de usuario en DTE Platform.
3. Completar perfil inicial (nombre y WhatsApp).
4. Crear primer proyecto.
5. Definir canal de comunicación y responsables.
6. Iniciar ejecución y calendarizar primer informe.

Cómo se crea un usuario en DTE Platform:
- Opción directa: ingresar a `/registro`, iniciar con Google y completar perfil.
- Opción por invitación de equipo: recibir link `/invite?token=...`, iniciar con Google y quedar vinculado al equipo cliente.

Después del alta, se configura el primer proyecto con objetivo, prioridad y equipo. Desde ese momento, el seguimiento se hace en la plataforma por módulos: Proyectos, Tareas, Informes, Facturas y Mensajes.

Resultado esperado del onboarding:
- usuario activo;
- proyecto visible en plataforma;
- canal de comunicación definido;
- plan inicial de trabajo con fechas y responsables.
