# Quiénes trabajan en DTE – equipo completo y especialidades

DTE trabaja con un modelo de equipo flexible: un núcleo estable que dirige y ejecuta, más una red de especialistas según el tipo de proyecto. Esto permite mantener calidad y velocidad sin sobredimensionar estructura. Para un cliente, esto se traduce en algo simple: siempre hay un responsable claro, y el equipo que se suma es el necesario para lograr el resultado.

El núcleo de DTE suele incluir roles de dirección de proyecto, estrategia comercial/performance, desarrollo web y automatizaciones. En proyectos típicos, hay una persona que lidera la estrategia (definición de oferta, embudo, prioridades), otra que coordina la ejecución (tareas, tiempos, entregables), y perfiles técnicos que implementan (front-end, back-end, integraciones, tracking). En paralelo, se suman perfiles creativos cuando el proyecto lo requiere: edición de video, diseño gráfico, guionado, motion, UGC, etc.

Especialidades frecuentes dentro de DTE (o en su red) incluyen: Paid Media (Meta Ads, campañas de captación y remarketing), CRO (optimización de conversión en landings y checkout), Web & UI (React/Next.js, performance, UX), Automatizaciones (WhatsApp, CRM, email, calendar, integraciones con APIs), Datos (dashboards, reportes, tracking), y Producción Creativa (piezas para anuncios, reels, contenido comercial). También puede haber perfiles de soporte como asistencia operativa, QA y documentación.

Cuando un prospecto pregunta “¿cuántos son?”, la respuesta correcta es: DTE opera como un equipo modular. Para un proyecto pequeño, pueden intervenir 2–3 personas. Para un proyecto grande, 4–8 personas entre estrategia, creativos y desarrollo. El cliente no compra “horas sueltas”; compra un sistema en funcionamiento, con un equipo armado alrededor de ese objetivo.
