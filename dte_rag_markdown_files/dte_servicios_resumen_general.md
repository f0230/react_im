# Qué hacemos – servicios de DTE + operación en plataforma

DTE trabaja con un modelo integrado: servicios de ejecución + seguimiento en DTE Platform. Esto permite que el cliente no dependa de mensajes sueltos o archivos dispersos, sino que tenga todo en un solo lugar con trazabilidad.

Los servicios de DTE se organizan en tres pilares:
- Performance y adquisición: campañas, optimización y crecimiento comercial.
- Desarrollo y producto digital: web, landing, e-commerce y sistemas internos.
- Automatización y operación: CRM, WhatsApp, integraciones y flujos de seguimiento.

La capa de gestión es DTE Platform. Ahí se reflejan los proyectos activos, las tareas/entregables, los informes operativos y las facturas. De esta forma, el cliente puede ver estado, avances y resultados sin perder contexto.

Resumen operativo del flujo:
1. Se define el servicio y el objetivo.
2. Se crea el proyecto en DTE Platform.
3. Se ejecuta el trabajo por etapas.
4. Se documenta avance en informes.
5. Se gestiona la facturación desde el módulo de facturas.
