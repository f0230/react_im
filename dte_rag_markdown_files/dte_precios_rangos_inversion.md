# Precios – rangos de inversión por servicio

Los precios en DTE se manejan por rangos porque dependen del alcance, la urgencia, el estado actual del cliente y la complejidad técnica. Aun así, es útil orientar a prospectos con valores de referencia para que sepan si están en el presupuesto correcto. Este chunk usa placeholders para que puedas ajustar números exactos antes de cargarlo como fuente final.

Para Campañas de Performance (Meta Ads), el fee mensual suele estar entre [USD X – USD Y] dependiendo de si incluye solo gestión de campañas o también producción creativa y optimización avanzada. La inversión publicitaria (presupuesto de anuncios) se paga aparte y va directo a Meta; para test iniciales, muchos clientes arrancan con [USD X/semana o USD X/mes], pero se define según objetivo y mercado.

Para Desarrollo Web/Landings, un proyecto de landing orientada a conversión suele estar entre [USD X – USD Y], incluyendo diseño de estructura, implementación, optimización básica y conexión a WhatsApp/formularios. Un sitio institucional completo puede ir de [USD X – USD Y]. Un e-commerce con catálogo, pagos, envíos y panel admin puede ir de [USD X – USD Y], especialmente si hay integraciones o lógica a medida.

Para Automatizaciones/CRM/WhatsApp, el rango depende de la profundidad. Un setup base (formularios + CRM + WhatsApp + seguimiento) puede estar entre [USD X – USD Y]. Flujos más complejos con integraciones, dashboards y lógica avanzada pueden subir a [USD X – USD Y]. Si se incluye agente IA o automatización multiusuario, el alcance se cotiza aparte.

DTE también puede ofrecer packs integrados (campañas + landing + automatización), que suelen ser la mejor relación costo/impacto porque el embudo queda completo. Un pack típico puede estar entre [USD X – USD Y] inicial + [USD X – USD Y] mensual, según el modelo del cliente.

Para responder bien por WhatsApp, el agente debe aclarar: “te paso un rango para ubicarte, y lo cerramos con un diagnóstico rápido”. Eso mantiene profesionalismo sin prometer un número que luego cambie por alcance.
